Petunjuk Setup:

  1. Buat Google Sheets baru atau gunakan yang sudah ada
  2. Buat sheet dengan nama: "income", "expenses", "members", "activities", "articles", "donations"
  3. Set sharing ke "Anyone with the link can view"
  4. Dapatkan API Key dari Google Cloud Console
  5. Enable Google Sheets API di project Anda

Jika tidak memiliki API Key, Anda dapat menggunakan mode demo dengan data sample.
